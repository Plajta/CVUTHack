### Instalace
1. Repozitář obsahuje velké soubory spravované `git lfs`

	```
	git lfs install
	```
2. Nainstalovat requirements (doporučeno do venv nebo Conda enviromentu)

	```
	pip install -r requirements.txt
	```

### Troubleshooting
Pokud máte problémy s načítáním modelů

```
git lfs pull
```
